Développement spécifique

Développement et maintenance

Développement et maintenance d'application Python 3 ou JavaScript . Développement possible sur un framework propriétaire, NativeScript ou Cordova . Ils perrmettent de livrer de manière flexible et rapide des logiciels spécifiques aux process et aux contraintes de chaque entreprise.

Le framework et les serveurs sont basés sur des technologies open source, ne nécessitant aucune licence payante pour leur exploitation. Des exemples de code sont ouverts sur la plateforme de développement GitHub.

Testing et qualité

Testing logiciel grâce à des outils comme Unittest, Pytest, Appium, Selenium, voir même de la librairie maison Pybot pour les cas difficiles. Ces outils permettent de tester des applications desktop, web, mobiles et embarquées de manière automatisée. Des serveurs Gitlab, SonarQube et Jenkins internes ou dans le cloud permettent l'intégration et la livraison continue.

Automation

Les libraires Selenium et Appium sont connues pour l'automatisation de tests. Elles rentrent dans le processus d'intégration continue de projet web et mobile de qualité. Des technologies basées sur le computer vision comme celles dérivées d'openCV, Sikuli et Lackey peuvent être utilisées. Elles rendent possible l'automatisation lorsqu'aucune balise n'est identifiable.

Documentation des librairies maison social et Pybot.

Elles permettent également l'automatisation d'une multitude de tâches pour lesquelles aucunes API web ou bot n'est disponible:

  • Création de cannaux de communication transverses à plusieurs réseaux sociaux.
  • Posts marketing sur les pages de prospects.
  • Récolte de données sur le web ou les mobiles.
  • Augmentation du traffic sur les réseaux sociaux et sites web.

Logistique

Solution logistique basée sur les API Google Map ou Open Source avec serveur géomatique. Paramétrage des dépôts, véhicules, des préférences Google Map, des articles. Portail client avec envoi des factures. Staffing et autostaffing des véhicules.

Accès aux services paramétrable pour les différents profils, gestionnaires administrateur et livreur. Application mobile pour les livreurs. Restitution des KPI intégrée dans le framework PapIT.

Optimisation de livraison par résolution du Travelling Salesman Problem, Vehicle Routing Problem et Open Vehicle Routing Problem.
