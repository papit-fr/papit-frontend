Développement

Réalisation et maintenance

Développement et maintenance de backends d'applications Python 3, CoBOL, Java ou JavaScript. La gestion de la donnée et tous les challenges qui vont avec sont maîtrisés grâce à une grande expérience des SGDB SQL (DB2, MySQL). Cette dernière a été acquise en maintenant des applications comptant parfois une bonne partie des Français comme utilisateurs.

Les développements sont, tant que possible, basés sur des technologies open source auditables et ne nécessitant aucune licence payante pour leur exploitation. Des exemples de code sont ouverts sur la plateforme de développement GitHub.

Testing et qualité

Testing fonctionnel de tous logiciels, API web, réseau, embarqué, etc, grâce à Pytest. Appium, Selenium, permettent de tester les IHM d'applications desktop, web, mobiles et embarquées de manière automatisée. SonarQube est utilisé pour produire des KPI sur la qualité du code et guider le testing. Les scripts de test développés et SonarQube sont intégrés dans des outils de CI/CD comme Jenkins pour faciliter la supervision de la qualité logiciel et leur livraison.
